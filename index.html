Digite uma mensagem para criptografar:

Mensagem criptografada:
A mensagem é primeiro convertida para o formato de codificação de caracteres ASCII e depois criptografada usando a equação Mensagem Criptografada = (Mensagem) E % N.

Descriptografar uma mensagem

Digite uma mensagem criptografada para descriptografar:

Mensagem Descriptografada:
A mensagem criptografada é primeiro descriptografada usando a equação Mensagem Descriptografada = (Mensagem Descriptografada) D % N e depois convertida para o formato de codificação de caracteres ASCII para recuperar a mensagem original.
